Windows下安装Hexo教程

Windows Hexo

阅读时长:   1 分

Hexo 是一个快速、简洁且高效的博客框架，本文笔者带大家安装 Hexo。

安装前提

  • Node.js:https://nodejs.org/
  • Git:https://git-scm.com/

    安装 Hexo

    npm install -g hexo-cli
hexo init <> //<>内填Hexo初始化目录，会自动新建目录。
cd <> //进入刚刚初始化的新目录
npm install //进行安装

    启动 Hexo

    完成了安装 Hexo 步骤之后，便可以在本地启动 Hexo 了。
    hexo clean //清除缓存
hexo g //生成静态文件
hexo s //在本地部署Hexo
    执行 Hexo s 之后，Hexo 便会在本地运行，浏览器输入 127.0.0.1:4000 便可看到效果。

部署 Hexo 到 Github 或 Coding

如需部署 hexo 到 Github 或 Coding，需要在 Hexo 根目录的_config.yml 文件内填写 Github 的仓库地址，并在 Git 中配置用户名及邮箱地址。

#Hexo执行清除缓存，生成静态文件，以及推送到Github。
Hexo clean&&hexo g&&hexo d
